# Configuration
## Sections
### Config
#### screen_width
The width of the game screen.  
Default is 1280.

#### screen_height
The height of the game screen.  
Default is 720.

#### music_volume
Current music volume level (0-100).  
Default is 50.

----
# Development Notes
## Example Formatting; with arrays, integers and strings

```
[Config]

level={
"current": 1,
"unlocked": 1,
"total": 6
}
music={
"current": 0,
"max": 100
}
screen_height=1280
screen_width=720

```
